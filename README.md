# 💼 GitHub Portfolio

A modern developer portfolio built using **React** that showcases selected GitHub repositories with custom UI, search, filtering, and project cards.

🔗 Live Demo: https://your-portfolio-link.com  
📂 GitHub: https://github.com/Atharva20041809

---

## 🚀 Features

- 🔹 Display selected GitHub repositories (not all)
- 🔹 Custom project cards with images
- 🔹 Search repositories by name
- 🔹 Filter by programming language
- 🔹 Responsive layout (mobile-friendly)
- 🔹 Clean GitHub-inspired dark UI

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (ES6+)
- CSS3
- GitHub REST API
